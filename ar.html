AR+ for Grade 4

Augmented reality (English augmented reality, AR — "augmented reality") is the result of introducing any sensory data into the visual field in order to supplement information about the environment and change the perception of the environment.

To install the augmented reality app for your textbook, you need to do a few simple steps.

1. Copy the apk file from the site digital.lidrary into your phone using a cable or via Bluetooth.
2. In the phone settings, allow the installation of programs from unreliable sources.
3. Find the downloaded apk file in your phone and run it.
4. Answer yes to the phone's security system request and complete the installation.
5. The program has appeared on your phone's desktop and you can run it.
6. To work, the program needs permissions to access the camera and phone memory. Permissions can be given at the first start of the program.
